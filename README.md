# ubuntu-install
Script for installing tools and dev apps on ubuntu

## In current version installs:

  * JDK and JRE
  * Google Chrome
  * Telegram
  * Vim
  * Git
  * Codeblocks
  * Atom
  * rpm
  * npm
  * nodejs
  * unity-tweak-tools

## Installation
```bash
git clone https://github.com/alikhil/ubuntu-install
cd ubuntu-install
# you cand edit script and remove apps that you don't need
chmod +x install.sh
./install.sh
```
